# DatingApp-Udemy-

#You need to add appsettings.json 
```
{
  "AppSettings" : {
    "Token" : "can be any secret key"
  },
  "ConnectionStrings": {
    "DefaultConnection":"your connection string.. Mine was SQL Server"
  },
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft": "Warning",
      "Microsoft.Hosting.Lifetime": "Information"
    }
  },
  "AllowedHosts": "*"
}
```
